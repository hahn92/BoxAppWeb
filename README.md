# MiApp - Sitio web estático

Este proyecto contiene una web simple con 3 páginas:

- `index.html` — Página principal con descripción, funcionalidades y enlaces de descarga.
- `privacy.html` — Política de Privacidad.
- `terms.html` — Términos de Uso.

Para probar localmente:

1. Abre un terminal y ve a la carpeta `site`:

   ```bash
   cd /Users/hahn/Documents/Desarrollo/BoxAppWeb/site
   ```

2. Arranca un servidor simple (Python 3):

   ```bash
   python3 -m http.server 8000
   ```

3. Abre `http://localhost:8000` en el navegador.

Personaliza textos, enlaces de tienda y correos electrónicos en los archivos HTML según tus necesidades.
